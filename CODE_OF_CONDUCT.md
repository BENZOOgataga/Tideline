# Contributor Code of Conduct

## Our standard

This project is meant to be useful, calm, and respectful. We want contributors, users, and maintainers to be able to participate without being talked down to, harassed, or dragged into personal fights.

By participating in Tideline spaces such as issues, pull requests, discussions, and review threads, you agree to behave professionally and respectfully.

## Expected behavior

- Be respectful in disagreement.
- Assume good intent, then ask for clarity when something is unclear.
- Keep critique focused on the work, not the person.
- Welcome newcomers and help them navigate the project norms.
- Respect privacy. Do not pressure people to share personal data, logs, or note contents publicly.
- Accept maintainership decisions, even when your proposal is declined.

## Unacceptable behavior

- Harassment, threats, stalking, or intimidation.
- Insults, slurs, or demeaning language.
- Sexual language or unwelcome sexual attention.
- Trolling, dogpiling, or deliberately inflammatory behavior.
- Publishing someone else's private information without clear permission.
- Repeatedly pushing the same argument across multiple threads after a decision has been made.

## Scope

This Code of Conduct applies in all project spaces, including:

- GitHub issues
- pull requests
- discussions
- review comments
- any project-linked chat or community space

## Maintainer responsibilities

Maintainers may edit, hide, lock, or remove comments, issues, discussions, commits, or pull requests that violate this Code of Conduct. They may also warn or block contributors whose behavior is not compatible with a healthy project environment.

## Enforcement

Possible responses include:

- a private warning
- removal of specific content
- temporary lock or suspension from project spaces
- permanent ban from project participation

The response will depend on severity, pattern of behavior, and impact on other participants.

## Notes

This document is intentionally short and project-specific. It is not a legal contract. It is a working standard for how people are expected to treat each other here.
